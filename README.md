## Getting Started

First Install Depedencies :

```bash
pnpm install
```
Start development server :
```bash
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## This App Using Package :
- NextJS 14
- NextAuth
- Typescript
- Shadcn UI
- Upstash redish
- Pusher JS
